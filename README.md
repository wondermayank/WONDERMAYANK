[**🌐 Live Demo**](https://wondermayank.github.io/)

---

## 🎨 Design Philosophy
Inspired by **VisionOS** and **Apple’s Design Language**, the interface focuses on:
* **Glassmorphism:** Deep blur effects and semi-transparent layers for a floating UI feel.
* **Bento Grid:** A clean, modular layout for high-density information.
* **Fluid Motion:** Custom cubic-bezier transitions and 3D hover interactions.
* **Minimalism:** A soothing Lavender-to-Sky-Blue gradient system.

---

## 🚀 Featured Projects
The hub provides instant access to several specialized tools:
* 🎬 **Anime & Movie Finders** - Interactive media discovery.
* 📖 **Books Hub** - Personal library explorer.
* 💌 **Card Generator** - Dynamic event card creator with sharing.
* 📄 **Utility Tools** - Text-to-PDF converters and coding helpers.
* 🎨 **Paletteify** - Aesthetic color palette generator.

---

## 🛠️ Tech Stack
* **Core:** HTML5, CSS3 (Modern Grid & Flexbox)
* **Styling:** Custom CSS Variables & `backdrop-filter` for glass effects.
* **Interactions:** Vanilla JavaScript (Intersection Observer & 3D Tilt Logic).
* **Typography:** Apple SF Pro Display / Inter.
